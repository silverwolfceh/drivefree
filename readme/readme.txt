Truy cập vào: https://console.cloud.google.com
Làm như hình:

B1: Tạo project mới (1.png)
B2: Đặt tên project tuỳ ý. Sau khi tạo xong chọn project vừa tạo và vào API & Service (2.png)
B3: Tìm từ khoá drive , click vào sau đó enable (3.png)
B4: Làm như hình đến ảnh 6.png khi tạo xong , sẽ tự động tạo và download file .json xuống
B5: Rename file .json đó thành "v1.json" sau đó paste vào trong localhost đặt cùng hàng với file index.php
B6: Tương tự có thể tạo thêm v2.json , v3.json để thêm dung lượng bằng các lặp lại từ B1
B7: Chạy thử index.php trong localhost

Lưu ý nên tạo folder rồi mới upload, ko up ở thư mục gốc
